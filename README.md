## How to install
``` sh
$ npm install
```

## How to build
``` sh
$ npm run build
```

## How to run
``` sh
$ npm start
```
Open you browser and visit [http://localhost:8080](http://localhost:8080).

## How to run in development mode
``` sh
$ npm run dev
```
